# Random Password Generator

A sleek, minimalist web application that instantly generates secure, random password options with a single click. 

## Features

- **Dual Outputs:** Generates two highly secure passwords simultaneously for quick selection.
- **Visual Contrast:** High-contrast design featuring vibrant accent text against a dark, distraction-free background.
- **Instant Generation:** Simple, straightforward user experience focused purely on getting a secure password with zero clutter.

## Tech Stack

- **HTML5:** Clean, semantic markup.
- **CSS3:** Modern dark-mode styling with Google Fonts integration.
- **JavaScript:** Dynamic string randomization and DOM manipulation.
